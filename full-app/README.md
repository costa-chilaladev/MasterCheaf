# 🍽️ PratoFacil — Full Application

PratoFacil is a full-stack recipe-sharing platform where users can create, explore, favorite, save, and comment on recipes.

This is the **main version** of the project, including backend functionality, database integration, and dynamic user interactions.

---

## ✨ Features

### 👨‍🍳 Recipe Management
- Create recipes
- Explore recipes
- Save favorite recipes
- Comment on recipes

### 🎨 User Experience
- Responsive interface
- Modern UI design
- Interactive components
- Dynamic recipe organization

### 🖼️ Media & Inputs
- Image cropping with Cropper.js
- Enhanced select inputs using Tom Select

---

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- PHP
- MySQL

### Libraries
- Cropper.js
- Tom Select
